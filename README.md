Trabalho acerca da evasão acadêmica, desenvolvido com o intuito de modelar e implementar um protótipo de código que permita: 
Identificar perfis com maior risco de evasão; 
Reconhecer padrões associados ao sucesso e ao fracasso acadêmico;
Discutir como essas estruturas podem apoiar ações de acompanhamento, prevenção ou intervenção
institucional.

O arquivo "GrafoEx.py" gera a representação visual do grafo já modelado dada uma porção limitada dos alunos.

O arquivo "DispersaoG.py" retorna uma gráfico que representa os clusters gerados pelo algoritmo ao processar a base de dados.

Por fim, o arquivo "HEAL-GNN.py" serve como prova de executabilidade do framework. Trata-se de um script mais leve do que o idealizado no artigo por limitações técnicas.
